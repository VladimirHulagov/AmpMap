Вкладка Settings
====================

На вкладке **Settings** возможно конфигурирование интервала времени, в
рамках которого доступно редактирование тестовых результатов:

- Изменение тестового результата в рамках указанного интервала времени
  (опция по умолчанию, значение — 1 час (1h);
- Редактирование тестового результата без ограничений по времени
  (значение оставлено пустым);
- Запрет на редактирование тестового результата.
