Вкладка Overview
================

На вкладке **Overview** доступен просмотр общих сведений о проекте и
их редактирование:

|projectOverview|

Чтобы отредактировать общую информацию о проекте, нажмите на кнопку редактирования проекта расположенную в правом верхнем углу приложения.
