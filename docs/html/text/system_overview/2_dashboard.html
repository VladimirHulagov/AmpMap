Раздел Dashboard

Раздел является стартовой страницей и содержит информацию обо всех существующих в системе проектах. По умолчанию проекты отображается в виде плиток, доступно переключение на табличное представление () .

На стартовой странице системы пользователь может выбрать язык интерфейса, а также один из трёх вариантов темы интерфейса: * Светлая * Тёмная * Системная

Пользователю доступен поиск по проектам. В левой части экрана расположено поле поиска для навигации в рамках системы:

Пользователь может добавить любой проект на стартовой странице в избранное с помощью кнопки — такие проекты показываются пользователю в первую очередь.

В карточке проекта отображается его название, аватар, краткая статистика и кнопки навигации ( Overview , Test Suites & Cases и Test Plans & Results ).

Нажав на название проекта, пользователь переходит на вкладку Overview, на которой представлена сводная информация по проекту, а также доступно администрирование проекта (администрирование проекта описано в разделе Администрирование проектов).

В правом верхнем углу страницы пользователю доступно добавление проекта в избранное, а так же кнопка настроек проекта, нажатие на которую открывает модальное окно для редактирования общей информации о проекте.
